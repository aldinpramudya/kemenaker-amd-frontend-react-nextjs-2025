## MAGANG KEMNAKER AMD ACADEMY GUIDE

1. Clone Project

```bash
clone https://github.com/aldinpramudya/kemenaker-amd-frontend-react-nextjs-2025.git
```

2. Instalasi Seluruh Package
```bash
npm install
```

3. Jalankan project
```bash
npm run dev
```

Buka di [http://localhost:3000](http://localhost:3000)
